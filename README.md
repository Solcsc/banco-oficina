# Banco Oficina

Este repositório contém os arquivos do projeto de banco de dados da oficina mecânica.

## 📂 Estrutura
- **scripts/** → contém o script SQL com a criação das tabelas e inserção de dados (`oficina.sql`).
- **diagramas/** → contém o diagrama relacional do banco (`modelo_relacional.png`).

## 🚀 Como usar
1. Abra o MySQL Workbench ou outro cliente SQL.
2. Execute o arquivo `scripts/oficina.sql` para criar o banco e popular com dados.
3. Consulte o diagrama em `diagramas/modelo_relacional.png` para visualizar a estrutura das tabelas e relacionamentos.

## 📌 Conteúdo
- Criação de tabelas (clientes, veículos, ordens de serviço, equipes).
- Inserção de dados de exemplo.
- Consultas SQL com filtros (`WHERE`), ordenações (`ORDER BY`), agrupamentos (`GROUP BY` e `HAVING`) e junções (`JOIN`).

## 🛠️ Tecnologias
- MySQL
- MySQL Workbench (para modelagem e diagrama)
